# Security Policy

## Reporting a Vulnerability

If you find a security vulnerability in this project, please report it
**privately** — do not open a public issue or pull request.

- **Preferred:** use GitHub's private vulnerability reporting — the
  **"Report a vulnerability"** button on the repository's
  [Security tab](https://github.com/danajerban/erbandanaj/security/advisories/new).
- **Alternatively:** email **danajerban@gmail.com** with a description and steps
  to reproduce.

I'll acknowledge your report as soon as I can and keep you posted on the fix.

## Scope

This is a personal portfolio site. The most useful reports concern issues that
could compromise the build/deploy pipeline or visitors to the live site. Thanks
for helping keep it safe.
